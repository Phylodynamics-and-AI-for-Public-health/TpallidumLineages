# _Treponema pallidum_ PopPUNK Lineages from Long _et al._ 2025

To use this database, please install [PopPUNK](https://github.com/bacpop/PopPUNK) and download the contents of this repository. You will need to ensure that **a)** the files are put in a folder called ```AllMasked``` or **b)** you rename the files to match whichever folder the database will be found. Also, please note that you will need to decompress ```AllMasked.h5.xz``` prior to using the database.
